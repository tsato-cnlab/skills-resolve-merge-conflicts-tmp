# GitHub Teacher

インターネット上の知識を、ガリレオが星を記録したように整理します。

## 職務経験

### GitHub Trainer

Git に関するあらゆることを教え、ステッカーを配り、世界平和を支えます。

### Supportocat

GitHub プラットフォーム上で、顧客に高品質なサポートを提供します。

## スキル

### 教育

Git、GitHub、オープンソースなど、さまざまなトピックに関するカンファレンストーク、オンライントレーニング、対面トレーニングを開発・維持しました。

### リーダーシップ

複数の非同期チームを管理し、さまざまな Web アプリケーションや Web サイトの開発、保守、リリースを進めました。
